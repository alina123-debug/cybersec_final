Playbook: Suspicious Windows Service

Investigation guide: check false positive first, then collect evidence, then contain/eradicate, then lessons learned.
1) False Positive check
  • Is it a legitimate software update/service?
  • Is the binary signed by a trusted vendor?
2) Evidence
  • service_name, service_path, start time
  • hashes + signer info
  • process tree + network connections
3) Example queries (5)
windows.service_install where service_name == <NAME> last 24h
edr.process where path == <PATH> and time within incident
edr.network where host == <HOST> and time within incident
registry.run_keys where host == <HOST> last 7d
hash.lookup where sha256 == <HASH>
4) Response
  1. Isolate host if confirmed malicious.
  2. Disable service, quarantine binary.
  3. Collect artifacts for IR report.
