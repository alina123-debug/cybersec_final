# CyberSec Monitor 360 (Django Edition)

A SOC-style dashboard with **real-time alerts/cases**, filters, playbooks wiki, rules, reports, and a log generator.

## Syllabus coverage (Advanced Programming - Python)
- Python basics, data types, collections, control structures
- Async (asyncio generator + WebSocket via Django Channels)
- Modules & files (project packages; notebook)
- OOP (Django models + service layer)
- SQLite (default DB)
- Web/REST (DRF)
- Django (templates + admin)
- Jupyter + NumPy + Pandas + Matplotlib (notebook)

## Quick start (Windows)
See **Deployment_Manual_Windows.pdf** in the 제출 ZIP.

## Demo flow
1) Seed data: 1 client, 5 responsible people, 5 rules
2) Start server
3) Run generator (creates ~75 alerts if you run with --count 75)
4) Open dashboard and watch live toasts + charts updates
